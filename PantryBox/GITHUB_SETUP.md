# How to Push PantryBox to GitHub from VS Code

## Step 1: Initialize Git Repository

1. Open VS Code
2. Open the PantryBox folder in VS Code
3. Open the terminal (Terminal > New Terminal)

```bash
# Navigate to your project directory
cd PantryBox

# Initialize git repository
git init

# Add all files to git
git add .

# Make initial commit
git commit -m "Initial commit: PantryBox Android app with Jetpack Compose"
```

## Step 2: Create GitHub Repository

1. Go to [GitHub](https://github.com)
2. Click the "+" icon in the top right and select "New repository"
3. Repository name: `luchbox` or `pantrybox`
4. Description: "Native Android app for pantry management and recipe suggestions"
5. Choose Public or Private
6. **DO NOT** initialize with README, .gitignore, or license
7. Click "Create repository"

## Step 3: Connect to GitHub Repository

```bash
# Add the remote repository (replace with your actual URL)
git remote add origin https://github.com/zikoa1234-ak/luchbox.git

# Verify the remote was added
git remote -v

# Push to GitHub
git branch -M main
git push -u origin main
```

## Step 5: Complete Push

```bash
# Add .gitignore
git add .gitignore
git commit -m "Add .gitignore"

# Push all changes
git push
```

## Step 6: Verify on GitHub

1. Go to your GitHub repository: `https://github.com/zikoa1234-ak/luchbox`
2. You should see all your files
3. The README.md should be displayed

## VS Code Git Integration

Alternatively, you can use VS Code's built-in Git tools:

1. Click the Source Control icon in the left sidebar (looks like a branch)
2. Click "Initialize Repository"
3. Stage changes by clicking the "+" next to files
4. Enter commit message
5. Click "Publish Branch" button (cloud icon)

## Troubleshooting

### If you get authentication error:
```bash
# For HTTPS (recommended)
git config --global credential.helper store

# Or use SSH instead:
git remote set-url origin git@github.com:zikoa1234-ak/luchbox.git
```

### If you need to force push (careful!):
```bash
git push -f origin main
```

### If you cloned instead of creating new:
```bash
# Remove the old connection
git remote remove origin

# Add your repository
git remote add origin https://github.com/zikoa1234-ak/luchbox.git

# Push
git push -u origin main
```
